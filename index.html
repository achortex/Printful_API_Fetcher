Printful API Fetcher

A powerful tool to fetch print templates and product mockups from the Printful API for your store products

Try Online Demo View on GitHub

Features

Template Fetching

Fetch and download print templates for your Printful products with ease. Get the exact specifications you need for your designs.

Mockup Generation

Generate high-quality product mockups directly from the Printful API. Visualize your products before production.

Product Management

Easily select and manage products from your Printful store. Streamline your product development workflow.

API Integration

Direct integration with the Printful API ensures you always have access to the latest product data and features.

Efficient Caching

Smart caching reduces API calls and improves performance, making your workflow faster and more efficient.

User-Friendly Interface

Intuitive design makes it easy to navigate and use, even for those new to Printful's API system.

Quick Start

Try the online demo at: https://printful.streamlit.app/

No installation required! Just visit the link above to start using the tool right away.

Installation

If you prefer to run it locally:

git clone https://github.com/achortex/Printful_API_Fetcher

Requirements

pip install -r requirements.txt

Usage

streamlit run app.py
